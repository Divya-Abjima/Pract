HTML stands for Hyper Text Markup Language. An HTML document usually consists of two parts : head and body. It is written in text editor like notepad. Format of an HTML document:
<DOCTYPE HTML>
<html>
<head>
</head>
<body>
</body>

To insert a paragraph in an html document paragraph tag is used. We can change the style of the paragraph by using its attributes like font and specifying the change.
Syntax: font style="arial",size=18
